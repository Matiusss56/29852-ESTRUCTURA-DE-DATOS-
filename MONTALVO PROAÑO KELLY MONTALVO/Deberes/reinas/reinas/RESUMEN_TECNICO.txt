┌───────────────────────────────────────────────────────────────────────────────┐
│ 8. FÓRMULAS Y NOTACIÓN MATEMÁTICA                                            │
├───────────────────────────────────────────────────────────────────────────────┤
│                                                                               │
│ Complejidad Temporal:                                                        │
│   T(n) = n × T(n-1)  donde T(1) = 1                                         │
│   T(n) = n! = n × (n-1) × (n-2) × ... × 1                                   │
│   Para n=8: 8! = 40,320                                                     │
│                                                                               │
│ Complejidad Espacial:                                                        │
│   S(n) = O(n²) + O(n) = O(n²)                                                │
│   Dominante: Memoria del tablero                                             │
│                                                                               │
│ Eficiencia de Poda:                                                          │
│   Ramas eliminadas = (Teórico - Práctico) / Teórico                         │
│   = (40,320 - 2,057) / 40,320                                               │
│   = 94.9% de ramas podadas                                                  │
│                                                                               │
└───────────────────────────────────────────────────────────────────────────────┘

┌───────────────────────────────────────────────────────────────────────────────┐
│ 9. DETALLES DE IMPLEMENTACIÓN                                                │
├───────────────────────────────────────────────────────────────────────────────┤
│                                                                               │
│ Acceso a Memoria:                                                            │
│   Sintaxis tradicional:        tablero[fila][columna]                        │
│   Sintaxis con punteros:       *(tablero + fila*8 + columna)                 │
│                                                                               │
│ Asignación de Memoria:                                                       │
│   int* tablero = new int[64];  // Asignar                                    │
│   delete[] tablero;            // Liberar                                    │
│                                                                               │
│ Validaciones:                                                                │
│   • esSeguro() - Verifica columna, diagonal izq, diagonal der              │
│   • Elimina ramas inválidas ANTES de explorarlas                            │
│   • Responsable del 94.9% de mejora de eficiencia                           │
│                                                                               │
│ Backtracking:                                                                │
│   1. Colocar reina                                                          │
│   2. Validar (esSeguro)                                                     │
│   3. Recursivamente siguiente fila                                          │
│   4. Descolocar reina (backtrack)                                           │
│   5. Intentar siguiente opción                                              │
│                                                                               │
└───────────────────────────────────────────────────────────────────────────────┘

│                                                                               │

│   O(log N)   → 3                    (Logarítmico)                            │
