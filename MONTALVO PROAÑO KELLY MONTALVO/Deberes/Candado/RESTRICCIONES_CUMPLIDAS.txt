===================================================

RESTRICCIONES ORIGINALES DEL USUARIO:
✓ No se puede utilizar vectores
✓ No se puede utilizar struct
✓ No se puede utilizar linked list
✓ No se puede utilizar [] (arreglos)

IMPLEMENTACIÓN FINAL:
====================

1. SIN VECTORES
   - No se usa vector<T>
   - No se usa STL containers

2. SIN STRUCT
   - No se define ningún struct
   - No se define ninguna clase personalizada

3. SIN LINKED LIST
   - No se usa Nodo*
   - No hay nodos con siguiente

4. SIN ARREGLOS []
   - NO se usa char[5]
   - NO se usa int[10]
   - NO se usa diccionario[i]
   - TODO se accede con aritmética de punteros
   - Formato: *(puntero + indice)

EJEMPLO DE TRANSFORMACIÓN:
==========================

ANTES (CON ARREGLOS):
  char password[5] = "1234";
  char temp = password[0];
  password[0] = password[1];

AHORA (SIN ARREGLOS):
  char* password = new char[5];
  *(password + 0) = '1';
  *(password + 1) = '2';
  char temp = *(password + 0);
  *(password + 0) = *(password + 1);

ACCESO A MEMORIA:
=================

Creación dinámica:
  char* p = new char[5];           ✓ Permitido (puntero)
  char arr[5];                      ✗ NO PERMITIDO (arreglo)

Acceso a datos:
  *(p + 0)                          ✓ Permitido (aritmética de punteros)
  p[0]                              ✗ NO PERMITIDO (notación de arreglo)

Comparación:
  *(p + i) == *(q + j)             ✓ Permitido
  p[i] == q[j]                      ✗ NO PERMITIDO
